# roman-numerals-kata
Kata for Roman Numerals

This kata consists of translating the decimal numeral system into the roman numeral system. Here are some examples of inputs and their expected outputs:

```
   1 ➔ I
   2 ➔ II
   3 ➔ III
   4 ➔ IV
   5 ➔ V
   9 ➔ IX
  21 ➔ XXI
  50 ➔ L
 100 ➔ C
 500 ➔ D
1000 ➔ M
```
